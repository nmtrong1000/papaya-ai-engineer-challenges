# PC-02. Create plan data model & seed file

## Requirement

A typed `plans.ts` data file exists at `src/data/plans.ts` containing the `BenefitType` enum, all TypeScript types, and the three plan objects from the challenge brief.

## Approach

Create `src/data/plans.ts` with the `BenefitType` enum (matching the four benefit keys in the brief), the `Benefit`, `Plan`, and `Plans` types from the ERD, and the three hardcoded plan objects. Use `Record<BenefitType, Benefit>` as the type for `benefits` so the enum drives iteration in the table. The `-1` sentinel and `null` values are kept as-is in the data; display logic in later tasks converts them.

- [x] Create `src/data/` directory
