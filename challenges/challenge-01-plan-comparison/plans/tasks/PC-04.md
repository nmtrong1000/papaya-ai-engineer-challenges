# PC-04. Render benefit rows (null → indicator, -1 → Unlimited)

## Requirement

`PlanCard` displays a fully rendered benefit section: `null` benefits show a styled "Not included" indicator, `-1` values display "Unlimited", and present values display formatted numbers with their field labels.

## Approach

Create `src/utils/formatBenefit.tsx` that exports a `formatBenefit(benefit: Benefit, type: BenefitType): React.ReactNode` function. For `null`, it returns a styled "Not included" node. For each `BenefitType`, it maps the relevant sub-fields, renders `-1` as `"Unlimited"`, and formats all other numbers with `toLocaleString()`. Add a benefit section to `PlanCard.tsx` that iterates `Object.values(BenefitType)`, calls `formatBenefit` for each, and renders the result beneath the existing scalar fields.
