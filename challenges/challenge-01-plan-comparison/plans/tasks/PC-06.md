# PC-06. Implement recommended badge calculation

## Requirement

Exactly one plan displays a "Recommended" badge, chosen by the highest value-for-money score computed from the plan data.

## Approach

Create `src/utils/recommended.ts` that scores each plan and returns the index of the winner. The scoring formula: `score = annual_limit / monthly_premium * (1 - copay_percentage / 100) * (1 / (waiting_period_days + 1))`. This rewards high coverage, low copay, and low waiting period relative to price — a defensible, transparent formula. Add a `RecommendedBadge` component (inline in `ComparisonTable.tsx` or a tiny `src/components/RecommendedBadge.tsx`) rendered in the plan's header cell when its index matches the winner. Style with a Tailwind pill (e.g. `bg-green-500 text-white text-xs rounded-full px-2 py-0.5`).
